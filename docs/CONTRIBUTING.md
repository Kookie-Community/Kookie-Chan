# Contribuindo com o projeto

## Como contribuir
Você pode contribuir com:
- Código
- Documentação
- Sugestões
- Correções
- Revisão de PRs

## Reportando bugs
Abra uma issue seguindo o template **[Bug Report](https://github.com/markelpher/KookieChan/blob/main/docs/Bug_Report.md)**:
- Explique o problema
- Inclua prints se possível

## Sugerindo melhorias
Use o template **[Feature Request](https://github.com/markelpher/KookieChan/blob/main/docs/Feature_Request.md)** e descreva:
- Motivação
- Solução desejada
- Ideias de implementação

## Enviando alterações
1. Faça um **[Fork](https://github.com/markelpher/KookieChan/fork)** do repositório
2. Crie um branch:
   ```
   git checkout -b minha-feature
   ```
3. Faça commits limpos  
4. Abra um Pull Request descrevendo as alterações

## Código de Conduta
Todos os membros devem seguir o **[Code Of Conduct](https://github.com/markelpher/KookieChan/blob/main/docs/Code_Of_Conduct.md)**.
